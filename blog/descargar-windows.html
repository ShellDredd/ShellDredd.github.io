Descarga cualquier producto oficial de MICROSOFT en 2 clicks.


Una de las pesadillas recurrentes de l@s informátic@s se basa en la tediosa labor de descargar productos oficiales de la empresa Microsoft, sobre todo versiones de sus sistemas operativos tanto domésticos como servidores. Los motivos para tener pesadillas son bastante sólidos dado que, la web oficial de Microsoft es muy poco intuitiva y prácticamente sin menus contextules, se basa en botoneras y redirecciones bastante absurdas desde cualquier punto de vista, ya sea el diseño de usuario o el de productividad a la hora de descargar desde la web. Otro punto en contra es la velocidad de descarga de algunos servidores usados para el almacenamiento, en ocasiones muy lenta, lo que provoca que una simple descarga de windows 10 sea eterna o incluso a veces, la descarga se corta y corrompe la ISO.
Pero por suerte, existe una web que hace scrapping a las webs de repositorios oficiales de Microsoft y que testea las descargas para siempre ofrecer las que mejor velocidad de descarga. En este artículo cotilleamos un poco sobre esta web y que en mi opinión es el santo grial para gestionar laboratorios o instalaciones de estos productos.


¿Cómo funciona esta plataforma web?


Una vez accedemos a la web, disponemos de un buscador en la parte superior y un menú muy intuitivo donde encontrar y filtrar rápidamente el tipo producto, su versión, edición, idioma y arquitectura. La variedad de productos es la misma que ofrece las webs oficiales de Microsoft, de hecho aquí están todos dentro del menu, algo que se pide desde hace años a la gran M dado que, los productos los tiene repartidos por diferentes webs y dominios, creando una confusión y pérdida de tiempo tremenda. En la web tb.rg-adguard.net han suplido todos estos problemas y lo hace muy bien, lo que la convierte en la preferida de cualquier informátic@.
A ver si aprendemos Microsoft.

Por ejemplo uno de los productos más difíciles de encontrar y descargar, son los plugins o packs de diversos productos, como máquinas virtuales, XboxOneUpdate, windows insider, Inbox App, Driver Kits (WDK), Microsoft Office, TechNet (Programa para Tester), entre otros. En definitiva, TechBench es una web perfecta para descargar productos de Microsoft y que no puede faltar en tu kit de herramientas para administración de logística.
